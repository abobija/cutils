# :hammer_and_wrench: cutils
C utilities

## Targets

Usage: `make TARGET`

| Target  | Description |
| ------------- | ------------- |
| `all`  | Build all |
| `test` | Run all tests |
| `clean` | Clean project |
| `estr`  | Build estr  |
| `estr.test` | Run estr tests |
| `cmder` | Build cmder |
| `cmder.test` | Run cmder tests |
